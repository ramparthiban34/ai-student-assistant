# AI Student Assistant

A simple AI Student Assistant web app built using HTML, CSS, and JavaScript.

## Features
- Topic summary generator
- Quiz question generator
- Study planner
- Task manager
- Dark/light mode toggle
- GitHub Pages ready

## Project Structure
- `index.html` - Main application file

## How to Run Locally
1. Download or clone the repository.
2. Open `index.html` in your browser.

## How to Host on GitHub Pages
1. Create a new **public** repository on GitHub.
2. Upload the project files.
3. Make sure `index.html` is in the root folder.
4. Open **Settings > Pages** in the repository.
5. Under **Build and deployment**, choose **Deploy from a branch**.
6. Select the `main` branch and `/root` folder.
7. Save the settings.
8. Wait for GitHub to generate your live site link.

Your hosted URL will look similar to:
`https://your-username.github.io/your-repository-name/`
